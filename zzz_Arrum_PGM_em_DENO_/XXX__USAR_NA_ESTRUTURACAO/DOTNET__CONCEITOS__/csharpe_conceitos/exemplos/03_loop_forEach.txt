conceito: forEach é uma membroDeLeitura em loop dos tipos que sao arrayColecao..com ele acessamos cada item da colecao e podemos fazer algo com cada item acessado a cada iteracao.

sintaxe: funcao forEach ( tipo item in colecaoAlvo) { faça algo acessando a varIteradora item que tem o item a cada iteracao }

especifidade_linguagem: diferente do js os loops nao sao desencadeados de arrays no C# o forEach no caso é uma funcao que recebe o array como argumento.

exemplo:
foreach(string item in colecao1) {
      Console.WriteLine(item);
    }
