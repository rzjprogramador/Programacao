# apresentacoes declaracoes

> atribuicao_e_reatribuicao:
var1 := "Rei" // primeira vez foi atribuido uso :=

var1 = "OutroNome" // segunda vez foi reatribuido com mesmo tipo de valor entao usa só  o igual "="

//obs: uma vez atribuido o tipo de valor não  pode ser mudado ..somente ser editado reatribuido com o mesmo tipo de valor ex: se foi atribuido como string tem que ser string ate o final ...

---
> encurtamento_de_declaracoes_de_tipos:

se tenho "mais que uma" prop que vai ser de um tipo posso declarar seus nomes e dizer um tipo só para todos 
ex: em params ::
( a, b, c string ) ao inves de ( a string,  b string, c string )
