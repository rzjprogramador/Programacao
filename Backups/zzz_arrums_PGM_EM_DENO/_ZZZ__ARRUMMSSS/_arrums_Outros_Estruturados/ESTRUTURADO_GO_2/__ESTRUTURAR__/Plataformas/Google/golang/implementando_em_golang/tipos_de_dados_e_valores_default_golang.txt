# TIPOS DE DADOS GOLANG

texto: string , valor_inicial: "" // é string vazia

numeroInteiro: int e uint 16 - 32 -64 o default é a arquitetura do sistema do pc do usuario., valor_inicial: é 0 "zero"

numeroDecimal: float  (32 -e- 64) o default é a arquitetura do sistema do pc do usuario., valor_inicial: é 0 "zero"

boleano: bool ( true e false ) o valor inicial do bool é false, valor_inicial: é false

erro: error , valor_inicial: nil // o valor inicial do tipo error é nulo nil em go.

---

valores_iniciais:
somente para var ao declarar e nao passar valor...ja na parte de inferencia pela logica ai ja se tem valor.
