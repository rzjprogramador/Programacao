# nvm:

tutorial: https://www.youtube.com/watch?v=XlcHff3khcI&t=309s
page: https://github.com/nvm-sh/nvm
instalar: pegue o comando curl ou wget
obs: feche e abra novamente o terminal pra fazer efeito.
nvm --help 
nvm ls-remote // vai no repo remoto online e lista todas versoes node disponiveis.
nvm install <versao completa> // vai instalar a versao que pediu do node.
node -v // vai evr a versao em uso do node.
nvm ls // ja vai no repo instalado na maquina e mostra as versoes disponiveis diferente do ls-remote que vai na web.
nvm use <versao já local ou poso colocar o alias que mostrou no nvm ls> // usar versao node ..posso usar tbm assim só com os 2 primeiros numeros da versao vai usar a mais recente ex: nvm use 19
